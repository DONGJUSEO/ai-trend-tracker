### Post-RSS 대안 전략
- **네이버 오픈 API**: `GET /v1/search/news.json?query=인공지능&sort=date` → originallink로 매체 필터링
- **소셜 리스닝**: 언론사 X(트위터) 공식 계정 모니터링
- **유튜브 자막 분석**: 뉴스 유튜버 영상 자막 → LLM 요약 → 텍스트 뉴스 변환

## 6. AI 기본법 (2026년 1월 시행)

### 핵심 규정
1. **고영향 AI**: 생명/안전, 채용/대출 심사, 생체 인식 → 특별 관리
2. **투명성/표시 의무 (제31조)**: AI 생성물에 워터마크/메타데이터 필수
3. **사업자 책무**: 데이터 편향 방지, 영향 평가, 기록 보관

### 앱 개발 시사점
- 채용 정보 추천이 AI 평가 기반이면 고영향 AI 분류 가능
- AI 생성 콘텐츠에 "AI 생성" 표시 필수
- EU AI Act 대비 처벌 수위는 낮지만 서비스 중지 권한 있음

## 7. 채용 API 상세

### 사람인 (Saramin) - MVP 최적
- **API**: `https://oapi.saramin.co.kr/job-search`
- **인증**: 개발자 센터에서 access-key 발급
- **파라미터**: keywords, job_mid_cd(IT=2), loc_cd(서울=101000), sort(pd=등록일순), count(최대110)
- **제한**: IP당 일 500회
- **문서**: https://oapi.saramin.co.kr

### 원티드 (Wanted) - 테크/스타트업 특화
- **API**: `https://openapi.wanted.jobs/v2`
- **엔드포인트**: GET /jobs, GET /companies
- **인증**: 파트너스 센터 제휴 신청 → Client ID/Secret 발급 (3-5 영업일)
- **장점**: Tag 시스템 정밀 타겟팅, 연봉/성장성 데이터

### 잡코리아 (JobKorea) - 접근 제한적
- 공공기관 위주 API 개방
- 민간 사용은 B2B 계약 필요
