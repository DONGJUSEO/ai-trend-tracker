# 로고 설치 안내

현대로템 로고를 다음 위치에 저장해주세요:

## 필요한 파일

1. **메인 로고 (PNG)**
   - 파일 경로: `web/static/images/hyundai-rotem-logo.png`
   - 권장 크기: 400px × 200px (투명 배경)
   - 용도: 로그인 페이지, 메인 헤더

2. **파비콘 (선택사항)**
   - 파일 경로: `web/static/favicon.png`
   - 크기: 32px × 32px
   - 용도: 브라우저 탭 아이콘

## 로고 파일 준비 방법

1. 제공하신 현대로템 로고 이미지를 저장
2. 필요시 배경을 투명하게 편집
3. 위의 경로에 파일 저장

## 현재 구현 상태

✅ 로그인 페이지에 로고 영역 추가됨
✅ 메인 레이아웃 헤더에 로고 영역 추가됨
⏳ 로고 이미지 파일 업로드 필요

로고 파일을 저장한 후 페이지를 새로고침하면 로고가 표시됩니다.
